2018年个人投资总结 - 又一次见证历史

发表于 2018-12-28
本文字数： 3.8k | 阅读时长 ≈ 3 分钟

现在12月28日应该是2018年最后一个交易日，整年大盘从3500+点跌到今天的 2400+点，基本上30%左右的上证跌幅，创业版从1900+跌到现在 1200+ 大概40%左右吧。10月份从中信证券迁移到了东方财富，现在融资融券账号还要一个月才能开通。刚把资本金补充完整。19年的2月年后就可以重新启用融资融券账户进行交易了。今年前面8个月一直都是持有融资仓位的，最后2个月是没有办法所以并没有满仓+融资。

全年的收益算下来应该是 - 15%左右。整个大盘平均跌幅 - 30%左右。同时持有的基金今年整体收益也在 - 25% 左右。相比较2017年大约50%的收益相比，今年确实很惨。从10月份迁移到东方财富，最大的原因是拥抱了互联网技术和平台，东财的用户体验要甩开传统券商，比如中信，好几条街。在收益率测算，新股打新，系统速度，APP质量，闲置资金理财，等等多个方面，当然佣金也给我从万2.5降低到了万1.8。种种因素，同时也是因为我持续购买东方财富超过了三四年，最近两年更是重仓持有。越来越懂得一个道理，就是买自己看好的公司，在自己熟悉的领域内进行投资，不要随便乱来。

不论是价值投资，投机，趋势，都是因为吸收适合自己的操盘方式，多做总结，多反思。短线，中线，长线，都有可取之处。做任何事情应该是一个海纳百川的过程，然后逐步淬炼出属于自己的操盘手法。同时也不能故步自封，要时刻保持警惕性和学习心态，谦虚的面对市场。2018年的这一场下跌，熊贯全年。以下是自己今年全年操盘股票的逐个总结和反思

亏损和失败操盘

  • 上峰水泥 -30%

这只股票一方面听人推荐了一下，是一个比较靠谱的老前辈，同时自己也反复确认， 日线，周线，行情，业绩等等，在今年2月份重仓买入，现在回头看就是买在了大趋势的最高点，大概11-12元的成本吧。

很多时候，就是自己割肉不及时，会导致超过20-30%以上的亏损。自己总是学不好 止损点的及时割肉 这个关键性的操盘手法。从上峰水泥，以及以前的潞安环能，从负面角度看，完全反应出了自己性格上的不灵活，以及喜欢坚持的一方面。当然从另一方面，在比如科大讯飞 100%业绩翻倍上，也从正面角度体现出，坚持到最后的优势吧。

今年最大的亏损就是这一只股票了。在反弹到9.00+元多的时候选择了割肉出来。30%的亏损吧。最大的感受就是，在自己确实不熟悉领域的时候，不要盲目作为自己主要的投资方向，最好的选择方式可能是清仓参与，甚至模拟仓都可以。更多的是判断和分析，而不是持有。

阶段性盈利和当前持续持有的股票

2017年全年就做了一只科大讯飞，基本上25-28的成本吧。后来50+逐步卖出，17年的业绩应该算50-100%之间的业绩波动吧。毕竟有持续持有的，我也算不了那么精细。

从2017年10月份就持续看好两只，一只就是东方财富，一只是乐普医疗。东方财富是因为2015年股灾之后赚到了30%左右盈利。后来就清仓了。去年重新买入，大概12左右的成本，一直持有到了现在。依然还是12.50元左右的成本，折腾了一年，在东财上基本上账面是亏损的。

  • 乐普医疗 80%

在2017年10月份左右吧。20元买入，在2018年4-5月份，大概38+以上全部卖出吧。因为 中信证券全部销户了，也懒得找具体的擦盘记录了 ，收益率在80%。重点不是收益率，而是购入本金不太多。所以不足以影响整体资金收益率太多。

从今年5月份40元高点，到最近踩雷，跌破到20元以下。希望乐普的领头人可以坚持过这一次困境，当股市和公司熬过来的时候，我还会再回来的。

  • 华兰生物 20% 阶段收益 目前账面亏损5%

华兰生物在乐普医疗卖出之后进行的买入，基本上30左右买入的吧。在35左右进行了卖出。因为持有的仓位不多。所以更多的是一种判断和分析，因为对华兰这个公司不是很熟悉，所以前期买入的并不多

后来迁移到东财之后，在31.58 和 31.80的位置上进行重仓。一直持有到 40+以上做了卖出。卖出仓位并不多。主要是希望可以在2018-2019两年时间里在华兰生物上可以获得超过50%的收益。虽然现在来看希望渺茫。但是这可能已经是 最适合自己的操盘方式了 ，不论市场怎么恶劣，要活下来的人，都有属于自己独立生存能力的人，而不是依靠外界因素波动而挣扎的那一类人吧。

最近华兰生物踩雷了。目前成本在33.50元左右吧。基本上账面是有亏损的。最多还是可以持有20%亏损的，所以跌到 28元左右，我会考虑加仓的。2019年，华兰的目标还是 50元以上吧。或者成本做到25元以下。也行。。。

  • 乐视网 账面亏损20%

这只股票比较特殊一些，本来买入了1000股，也就是 3.55的成本，目的其实很单纯，希望站在同为创业者的角度，支持一下贾跃亭老先生。毕竟家里好几台电视还是乐视的，同时当年乐视辉煌的时候，也收入超过30%以上的收益，当年还是30+股价的年代，现在就10倍下来了。

后来卖出了900股，只保留了100股。当然现在只有 2.50的价格了。300多元RMB，博弈的不是钱的问题，而是一种信念。不论怎么困难，相信只要能坚持到底，时间会让一切的坏运气都熬过去。Winter is Coming ，最近一口气看完了 权力的游戏前面8个季 ，据说2019年1月份要上最后的最终季了。希望这一切都会过去。

也在这里，用微薄的一份心意，希望贾跃亭老先生可以熬过去。同为山西老乡，又是你曾经的既得利益者，也是你的产品消费者，还是你现有的某种意义上的股东。加油吧。

  • 恒生电子 账面亏损 10%

在今年6-8月之间，恒生电子进入了视野，模模糊糊中，感觉恒生做的应该是金融业的技术支撑，仿佛和马云，支付宝等都有一些关系。在这里没什么可分析的。主要是交易策略。

在46-48之间进行了买入。后来50左右做了卖出。期初基本上是一种试错的过程中。因为资本有限，所以恒生电子一直作为储备股票进行持有。后来在56的位置上购入了底仓，一直持有到了现在。2019年的希望收益率在30%左右。相信还是可以的。在融资仓位开启之后，希望恒生今年最后一根大阴柱之下，可以在1-2月份见底位置比如40-45之间均可以。到时候会考虑进行仓位上的加大。

百转千回的你

  • 东方财富 -10-20%

不知道应该怎么描述。去年非常看好，在12元位置买入。今年冲到15吧。当然我的希望肯定是翻倍收益，所以目标卖出价格在20元以上。今年到现在又回到了12元的位置，在9月份有一次大阴柱，这里就要 重点分析一下了 。在大阴柱前一个交易日，我敏感性的卖出了70%的仓位，在12.50附近吧大概，然后一口气跌到了10元的位置，我虽然看似聪明的及时进行了撤出，避免了大面积的亏损，但是在跌到10元的时候，也没有胆量进行抄底 ，后来从10元拉升到14元的过程中。自己基本上没有什么太好的收益。

后来转移到东财自己的券商里面，逐步仓位开始增加，来回几次冲上14，回落12的过程中。在12.50-14.00这个波动范围内。就这样折腾过了这一年。

目前成本在12.70左右。仓位也还行，也算比较高的。

本来预计2018年可以拿走30%-50%的收益，没想到一年到头来，账面上还是亏损的。因为在中信证券迁移的时候其实是净亏损了一部分，所以实际上东财今年的收益在-10%-20%之间吧。具体算不清楚了。

2019年，在这一波调整结束之后，价格可能会在10元附近吧。会加大融资仓位。 希望可以顺利持有一直到牛市天空之上，在若干年之后。。。。 ，因为自己也在使用它们的产品。所以还是在投资领域，和产品使用上，比较认可吧。

基金账面的收益

2017年大概15%的整体收益，因为所谓的白马股以及各种价值投资，到了2018年，全年亏损在25%左右。基本上全部歇菜。但是持有的几只基金还是很不错的。基金是从2017年家里的某种原因开始买入和持有的。持有目标是20年。基于家庭某种原因，所以会长长期持有 ，基本上会在2019年的阶段低位继续进行加仓。

目标是20年后，给 下一代留一份回报 ，简单说，赌国运了。

总结和2019年的展望

在写下2018年总结最后的时候，东方财富竟然没有去补这个11.83左右的缺口，匪夷所思的冲上了12.25左右。真是一个反人类的走法。1月份应该会再次回到缺口位置处理干净这点麻烦吧。当然可能下午一个跳水运动就都解决了。

2018年的全年账面收益，基本上在自己的预期之内，只有上峰水泥实际进行了亏损和卖出。其他大多数都属于阶段性盈利和账面暂时亏损的状态，而且账面亏损比例都在5-10%之间。

这一年从用了多年的中信，换到了东财，希望会是一个新的开始。

2019年，全年目标收益是以现在全部资本金的 50%的收益率。

重点观察

Cmder的使用

在win10下用传统的cmd的窗口虽然说也还可以，但是平时用习惯了linux下的操作方式，比如 ls ll等菜单，在win下会不太适应。那么Cmder这个神器应运而生。软件在Cmder 官方网站 下可以下载到。就是下载速度实在是不敢恭维。在此提供一份 Latest Version: v1.3.11 版本的 Full with Git的百度下载地址

以下是引用作者 晚晴幽草 在简书上的介绍

http://cmder.net/（她把conemu，msysgit和clink打包在一起，让你无需配置就能使用一个真正干净的Linux终端！她甚至还附带了漂亮的monokai配色主题。）;作为一个压缩档的存在, 可即压即用。你甚至可以放到USB就可以虽时带着走，连调整过的设定都会放在这个目录下，不会用到系统机码(Registry)，所以也很适合放在Dropbox / Google Drive / OneDrive共享于多台电脑。

下载的时候，有两个版本，分别是mini与full版；唯一的差别在于有没有内建msysgit工具，这是Git for Windows的标准配备；全安装版 cmder 自带了 msysgit, 压缩包 23M, 除了 git 本身这个命令之外, 里面可以使用大量的 linux 命令；比如 grep, curl(没有 wget)； 像vim, grep, tar, unzip, ssh, ls, bash, perl 对于爱折腾的Coder更是痛点需求。

网站部署说明

网站使用Hexo 纯静态博客程序 + Next 主题 最新的 6.0 版本进行构建。在本地进行安装，配置和调试工作。目前的使用方式是使用 hexo yuque 插件，在语雀编辑器上进行文章编辑，因为这个编辑器用起来的感觉是最舒服的。

语雀编辑器的使用

现在的文章编写之后，需要在本地进行文章拉取，然后构建，测试和发布，听说12月24日开始语雀开放全平台的webhook的方式，那么应该可以做到，语雀发布文章之后，触发webhook，然后推送到Git平台，然后通过Cl的持续构建，比如Jenkins或者其他方式，进行云端构建，然后通过YML的脚本，进行自动发布和部署。

Next 主题的使用

原来在使用wordpress的时候，在Github上用过一段时间的Next主题的改造版本，留下了很深的印象。现在开始使用Hexo之后，发现Next 仿佛是最受欢迎的主题之一。而且在初步调试和使用过程中发现确实很不错。当然开始是找到了5.0版本的，因为搜索到了中文的文档，后来发现最新的已经是6.6版本的，在Github上。然后在5.0升级到 6.0的过程中，遇到了一个大坑。就是无法进行直接升级，必须进行主题替换，然后_config.yml 文件进行修改和重新配置才可以，并不能直接升级，好尴尬。
